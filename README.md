# Resume Screening & Ranking System

## Project Overview

This project automates resume screening and candidate ranking using Natural Language Processing (NLP) and Machine Learning techniques.

The system analyzes resumes, matches them against job requirements, calculates candidate scores, identifies missing skills, and ranks candidates based on their suitability for a specific role.

---

## Features

- Resume text cleaning and preprocessing
- TF-IDF feature extraction
- Resume category classification
- Logistic Regression model
- Naive Bayes model
- Resume-to-job matching
- Candidate scoring system
- Candidate ranking based on role fit
- Skill gap identification
- Resume prediction demo
- Accuracy comparison
- Candidate ranking visualization

---

## Dataset

Resume Screening Dataset from Kaggle

Dataset Size:

- 962 resumes
- 25 job categories
